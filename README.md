# Shop-API

A RESTful Shop API (Node & MongoDB)

## Routes

- /products
- /orders

## Operations

### Products

Endpoint : /products

    - GET
    - POST

Endpoint : /products/id

    - GET
    - POST
    - DELETE

### Orders

Endpoint : /orders

    - GET
    - POST

Endpoint : /orders/id

    - GET
    - DELETE

## Dependencies

- express
- mongoose
- morgan

## Commands

- `npm run start`
